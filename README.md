<h1>구글 드라이브에서 파이썬 코딩하기 - 'Colaboratory'</h1>
<br/><br/>

파이썬 프로그래밍을 할 때, 별도의 IDE 없이 웹에서도 쉽게 코드를 작성하고 실행해볼 수 있는 환경이 갖춰져 있다.
특히 구글이 제공하는 클라우드 저장소인 'Google Drive'의 'Colaboratory' 앱을 이용하면 구글 드라이브에 파이썬 프로그램을 작성해 저장하고, 실행까지 할 수 있다.
